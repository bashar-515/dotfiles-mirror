```

```
bash etc/scripts/stow.sh
```

bash etc/scripts/macos.sh
```

## Settings

### System

- [P] Swap CTRL and CAPS LOCK keys: Settings -> Keyboard -> Keyboard Shortcuts -> Modifier Keys...
- [P] Stop automatically adjusting brightness: Settings -> Displays -> toggle off "Automatically adjust brightness"
- [P] Don't autofill passwords: Settings -> Autofill & Passwords -> toggle off "Autofill Passwords and Passkeys"

- Stop adjusting keyboard brightness in low light: Settings -> Keyboard -> toggle off "Adjust keyboard brightness in low light"
- Turn down keyboard brightness: Settings -> Keyboard -> turn down "Keyboard brightness"
